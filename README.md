Projeto da aula 01:

  * Inicialmente, um LED piscando constantemente em intervalos de 1 segundo

  * Apertando o botão 1, o intervalo é reduzido em 100 ms

  * Apertando o botão 2, o intervalo aumenta em 100 ms
  
  * Ao apertar os dois botões em um intervalo de 500 ms, o LED para de piscar
